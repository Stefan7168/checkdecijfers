# AGENTS.md

**The working agreements for this repo live in [CLAUDE.md](CLAUDE.md). Read that — it is the single source of truth.**

This file exists only so agent tools that look for `AGENTS.md` (instead of `CLAUDE.md`) are pointed to the real instructions. **Do not copy CLAUDE.md's content back into this file** — a second copy silently drifts out of date, which this project explicitly treats as a bug (see CLAUDE.md → *"Source of truth → Doc freshness — no stale docs"*). Keep this a one-line pointer; to change the rules, change [CLAUDE.md](CLAUDE.md).

(There is also a nested [web/CLAUDE.md](web/CLAUDE.md) for the web app — same convention, scoped to that directory.)
